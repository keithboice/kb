# @kb packages mono

<br />

## Getting Started

### Install

```bash
% git clone git@github.com:keithboice/kb.git

% cd kb

% npm install
```

### Develop

```bash
% npm run dev
```

### Test

```bash
% npm run test
```

### Documentation

```bash
% npm run docs
```

### Production

```bash
% npm run start
```
