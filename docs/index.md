# kb v0.0.1

## Getting Started

### Install

```bash
% git clone git@github.com:keithboice/kb.git

% cd kb

% npm install
```

### Develop

```bash
% npm run dev
```

### Test

```bash
% npm run test
```

### Flow

```bash
% npm run flow:coverage
```

### Documentation

```bash
% npm run docs
```

### Production

```bash
% npm run start
```
